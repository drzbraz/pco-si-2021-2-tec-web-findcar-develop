# FINDCAR

`SISTEMAS DE INFORMAÇÃO`

`TECNOLOGIAS WEB `

`2° SEMESTRE 2021`

O FindCar funcionará como um buscar de carros, para pessoas interessadas em praticidade na hora de escolher seu veículo, seja para aluguel ou compra. Busca o veículo desejado através de diversos filtros, em sites de anúncios específicos, em seguida redireciona o usuário para o anúncio original, permitindo o contato com o vendedor.
